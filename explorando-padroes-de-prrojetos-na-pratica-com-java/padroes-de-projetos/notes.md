# Padrões de Projeto

### Definição

* São soluções consolidadas para problemas recorrentes no desenvolvimento e manutenção de software orientado a objetos

### Classificações

* **Padrões criacionais**

  * Abstract factory

  * Builder

  * Factory method

  * Prototype

  * Singleton

* **Padrões comportamentais**

  * Chain of responsibility

  * Iterator

  * Observer

  * Strategy

  * Template method

* **Padrões estruturais**

  * Adapter

  * Bridge

  * Composite

  * Decorator

  * Facade

  * Flyweight

  * Proxy
